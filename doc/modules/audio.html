Module audio

Audio class

Functions

input_level (channel, level) set level for one input channel
output_level (level) set level for *both* output channels
monitor_mono () set monitor mode to mono both inputs will be mixed to both outputs
monitor_stereo () set monitor mode to stereo each input will be monitored on the respective output
monitor_on () enable monitoring (may cause clicks)
monitor_off () disable monitoring (may cause clicks)
pitch_on () enable input pitch analysis
pitch_off () disable input pitch analysis (saves CPU)
restart () restart the audio engine (recompile sclang)
